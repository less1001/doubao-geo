# Doubao Source Collector

一个最小可用的Chrome扩展，用来抓取豆包页面里展示的信息来源并本地保存。

## 当前能力

1. 自动监听页面变化
2. 识别包含“参考、来源、资料、搜索、引用”等关键词的区域
3. 提取其中的链接
4. 存入`chrome.storage.local`
5. 支持手动抓取当前页
6. 支持导出JSON和CSV
7. 支持清空本地记录

## 安装方式

1. 打开`chrome://extensions`
2. 开启“开发者模式”
3. 选择“加载已解压的扩展程序”
4. 指向当前目录

## 使用方式

1. 打开豆包网页并登录
2. 先正常提问一次
3. 等回答区和来源区都加载出来
4. 点击浏览器右上角扩展图标
5. 点“当前页抓取”
6. 看到“已抓取当前页来源”就说明保存成功
7. 需要整理文件时，点“导出Markdown”或“导出CSV”

## 输出内容

每条记录会尽量保存这些字段：

1. 问题
2. 回答
3. 来源
4. 来源网站
5. 作者
6. 日期
7. 来源链接
8. 来源上下文

## 说明

这个版本不依赖Playwright、Tampermonkey或Python。
后续如果需要自动测试、批量回放、离线整理，再单独加这些工具。
